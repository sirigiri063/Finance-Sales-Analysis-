# Project Structure

```
Finance-Sales-Analysis/
│
├── 📂 data/
│   ├── Finance_cleaned_dataset.csv     # Cleaned source data (CSV)
│   └── Financial_working.xlsx          # Excel working file with analysis
│
├── 📂 reports/
│   └── Sales_Financial_Analysis.pbix   # Power BI interactive dashboard
│
├── 📂 docs/
│   └── STRUCTURE.md                    # This file
│
├── 📂 assets/
│   └── dashboard_preview.png           # Dashboard screenshot
│
└── README.md                           # Project overview and usage guide
```

## Folder Descriptions

| Folder | Contents | Purpose |
|--------|----------|---------|
| `data/` | CSV + Excel files | Cleaned source data — connect to Power BI or explore in Excel |
| `reports/` | `.pbix` file | The Power BI dashboard — open with Power BI Desktop |
| `docs/` | `STRUCTURE.md` | Project structure documentation |
| `assets/` | Dashboard screenshots | Preview images used in the README |
