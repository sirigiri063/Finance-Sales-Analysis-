# 📊 Sales & Financial Performance Analysis Dashboard

## Overview
This project analyzes sales and financial performance data using Excel and Power BI. The dashboard provides interactive visualizations and business insights to help stakeholders understand revenue trends, profitability, product performance, and country-wise sales distribution.

## Objectives
- Analyze overall sales and profit performance.
- Identify top-performing countries and products.
- Track key business KPIs.
- Evaluate profit margins across different products.
- Support data-driven decision-making through interactive dashboards.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Units Sold
- Profit Margin %
- Country-wise Performance
- Product-wise Performance

## Key Insights

### 1. Highest Profit by Country
France generated the highest profit of **$3.78M**, contributing significantly to overall business profitability.

### 2. Best Product by Profit Margin
Amarilla achieved the highest profit margin percentage of **15.86%**, making it the most profitable product relative to sales.

### 3. Sales Performance Analysis
The dashboard highlights revenue trends, profitability patterns, and product performance across different markets.

### 4. Interactive Reporting
Users can filter and analyze data by:
- Country
- Product
- Segment
- Time Period

## Dashboard Features
